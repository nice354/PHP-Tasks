## Задание 1 — Карта сайта-ресторана

Я выбрал сайт **Dodo Pizza** (dodopizza.ru) — удобно, что там есть и меню, и корзина, и отслеживание заказа, так что примеров хватает.

| Фронтенд (вижу я) | Бэкенд (происходит на сервере) |
|---|---|
| Меню с фотографиями, ценами и описаниями | Запрос к базе данных за актуальным меню и ценами |
| Анимация при добавлении товара в корзину | Сохранение состава корзины в сессии пользователя |
| Форма ввода адреса с автоподсказками | Проверка адреса через геосервис, расчёт зоны доставки |
| Статус заказа обновляется без перезагрузки | Обновление статуса в БД, отправка push/SMS уведомлений |
| Фильтрация меню по категориям | Аутентификация, хранение истории заказов |

**Где JavaScript, а где Python?**

JavaScript работает на стороне браузера — это всё, что реагирует на действия пользователя без перезагрузки: анимации, валидация форм, обновление корзины, подсказки при вводе адреса.

Python (или другой серверный язык) — на бэкенде: обработка заказов, работа с базой данных, расчёт стоимости доставки, интеграция с платёжными системами и отправка уведомлений.

## Задание 2 — Детектив веб-разработчика

Сайт: **VK (vk.com)**

| Действие | Где обрабатывается | Почему |
| Нажал "Показать ещё комментарии" | Бэкенд | Браузер отправляет запрос, сервер достаёт следующую порцию комментариев из БД |
| Поставил лайк на пост | Бэкенд | Счётчик хранится на сервере — запрос уходит в БД, число обновляется |
| Поле ввода краснеет при ошибке | Фронтенд | Валидация и стили применяются прямо в браузере, сервер не нужен |
| Лента подгружается при скролле | Бэкенд | Новые посты приходят с сервера через API (infinite scroll) |
| Открытие выпадающего меню | Фронтенд | Чистое CSS/JS-взаимодействие, сервер не задействован |

## Задание 3 — Мой будущий проект

**Тема:** Платформа для онлайн-заказа и доставки продуктов питания

**Аудитория:** широкая — все, кто хочет заказывать продукты с доставкой на дом

**Страницы:**

- Главная страница с каталогом продуктов
- Страница категории товаров
- Страница отдельного товара (фото, описание, цена)
- Корзина и оформление заказа
- Страница отслеживания заказа
- Регистрация и вход
- Личный кабинет (история заказов, адреса доставки)
- Редактирование профиля

**Что смогут делать пользователи:**

- Регистрироваться и входить в аккаунт
- Просматривать каталог с фильтрацией по категориям
- Искать товары по названию
- Добавлять товары в корзину и оформлять заказ
- Указывать адрес и выбирать время доставки
- Оплачивать заказ онлайн
- Отслеживать статус доставки в реальном времени
- Смотреть историю заказов и повторять их
